# 画幅与布局对照表（单一来源）

所有格子数量对应的推荐画幅比例和布局方式参考。布局方式仅供参考——实际创作时应追求**手绘感的不规则排布**，格子位置自由错位、大小不一，避免机械网格对齐。人类手绘的漫画往往格子是"随意贴"上去的，而非完美的几何分割。

| 格子数 | 布局参考 | 推荐比例 | 说明 |
|--------|---------|---------|------|
| 2格 | 左右错落 | 2:1 | 宽条形，左右两格、上下轻微错位，分镜线可略微倾斜 |
| 2格 | 上下倾斜 | 1:2 | 长条形，上下两格左右错开，像随手贴上去的感觉 |
| 3格 | 错落三角 | 4:3 | 左上大格、右下两小格斜向排列，整体呈自然流动的布局 |
| 3格 | 斜向阶梯 | 3:1 | 超宽条形，三格从左下到右上呈阶梯状排列，每格位置略有偏移 |
| 4格 | 手绘散落 | 1:1 | 正方形画幅，格子像手绘草稿般散落排列、位置自由、有的略偏左有的略偏右，完全不追求对称 |
| 4格 | 斜向错落 | 4:1 | 超长条形，四格从左到右位置交替上下偏移，分镜线可轻微倾斜 |
| 6格 | 自然散落 | 2:3 | 竖长画幅，格子自由分布——有的偏左有的偏右，大小交替，像手账涂鸦 |
| 9格 | 手绘网格 | 1:1 | 正方形画幅，九宫格为参考但大幅打破——格子位置随机偏移，有的重叠、有的留出大空白 |

## 选择建议

- **手机近看**：优先 1:1（4格田字）或 3:4，格子大，字号清楚
- **PPT 投影远看**：优先 2:1 或 3:1，横向展示空间充足
- **海报打印**：优先 2:3 或 1:2，竖向排版视觉重，内容容量大
- **公众号**：优先 1:1 或 3:4，手机端兼容性好
