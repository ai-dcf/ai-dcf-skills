# 风格：复古印刷风

适用场景：怀旧内容、历史回顾、复古主题、经典文学、老报纸/杂志风格内容。具有年代感和故事感。

## 风格特征

- **质感**：模拟老式印刷品质感，纸张有轻微泛黄或纹理，油墨边缘有轻微晕染或不均匀感，可见细微颗粒感
- **色调**：复古色调为主，如 sepia（ sepia 褐色调）、老报纸黄、褪色红蓝、低饱和度暖色；避免高饱和霓虹色
- **氛围**：年代感、故事感、温暖怀旧，像翻阅旧杂志或老照片

## 分镜线

复古风格的分镜线，可用虚线、双线或做旧效果的实线；分镜线可略微不规则，模拟印刷品的时间痕迹；格子位置可自由错位，增强手绘排版感。

## 文字呈现

复古印刷字体或打字机字体风格，可有轻微油墨晕染效果；文字框可用老式对话框或报纸标题样式；字号适中偏大，确保清晰可读。

## 负面约束（阶段4写入 prompt）

不要现代扁平矢量感、不要3D渲染、不要高清摄影写实、不要科技感发光效果、不要日式漫画粗黑线条、不要水彩透明晕染、不要卡通圆滑风格、不要任何现代水印/二维码/数字标识、不要高饱和度鲜艳色彩。保持复古印刷品的年代质感。
