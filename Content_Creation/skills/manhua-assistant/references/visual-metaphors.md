# 视觉隐喻库

生成漫画提示词时参考此库。每个场景选择最直观、最大众的隐喻，避免使用抽象或行业术语。

## 核心情感与状态

| 场景 | 推荐隐喻 | 备选隐喻 |
|------|---------|---------|
| 超载/规则太多 | 塞爆的背包、溢出的纸条 | 仪表盘全爆红、桌上堆成山的文件 |
| 复杂→简单 | 打结毛线 → 整齐卷好的线团 | 一团乱麻 → 分步骤的小卡片 |
| 困惑→理解 | 角色看一本打开的书，脑袋上有"?" | 角色突然露出恍然大悟的表情，头顶亮起灯泡 |
| 失败→成功 | 角色困惑→思考→行动→喜悦（表情递进） | 角色从掉在地上站起来，脸上自信的笑容 |
| 成长/进步 | 角色站在阶梯上，每级台阶标记一个要点 | 小树苗→大树（根基深、树冠丰富） |
| 危险/风险 | 角色走在悬崖边，脚下是深渊 | 路牌上画着警告标志 |
| 安全/稳定 | 角色站在坚固的堡垒里 | 角色撑着大伞，雨水顺伞面滑落 |

## 工具与能力

| 场景 | 推荐隐喻 | 备选隐喻 |
|------|---------|---------|
| 新工具/技能 | 工具箱打开，里面整齐陈列工具 | 角色戴上工牌，手里拿着新工具 |
| 知识/学习 | 积木堆叠（从单块到塔） | 角色翻开书本，金色光芒从书页飞出 |
| 团队合作 | 多个角色一起推动齿轮 | 角色们牵手形成链条 |
| 自动化/流水线 | 传送带上物品顺序流动 | 齿轮咬合，顺畅运转 |

## 对比与冲突

| 场景 | 推荐隐喻 | 备选隐喻 |
|------|---------|---------|
| 对比冲突（错误vs正确） | 左边灰暗拥挤杂乱 → 右边明亮整洁 | 左边角色愁眉苦脸 → 右边角色轻松愉快 |
| 选择/决策 | 路口分叉，两条路各贴标签 | 天平左右放置两个选项 |
| 变化/转折 | 角色站在变化的路口，身后是旧路，前方是新路 | 季节变化：冬天→春天（环境变暖、绿芽生长） |

## 流程与结果

| 场景 | 推荐隐喻 | 备选隐喻 |
|------|---------|---------|
| 流程步骤 | 角色沿路牌依次前进，每个路牌一个步骤 | 锅里煮东西，每步往里加一个原料 |
| 目标/终点 | 角色到达山顶，远处是全景 | 角色拿到奖杯/勋章 |
| 结果展示 | 角色站在完成的作品前，表情满意 | 展示台上摆放最终成果，周围有星星/光芒 |
