# 常见问题与解决方案

生成漫画过程中遇到的高频问题及对策。主要用于阶段4提示词封装和阶段5迭代润色时参考。

## 风格跑偏

| 问题现象 | 根本原因 | 对策 |
|---------|---------|------|
| 生成日式漫画风 | 模型看到"comic/漫画"会触发强默认先验 | 在 prompt 中明确写"以奶油纸彩铅水彩风格为唯一基础，不得换风格"；加负面约束"不要日式漫画风" |
| 纸质感和笔触消失 | 可读性约束压过了质感细节 | 补充可观察特征："纸纹可见、彩铅笔触可见、轻晕染"，而不仅写"彩铅水彩" |
| 风格被当作可选项 | 风格词不够"排他" | 写"这是唯一允许的风格，不能切换到其他风格" |
| 变成扁平矢量/3D/摄影 | 风格词太短，模型自行选择 | 负面约束列出所有不要的风格：扁平矢量、3D、摄影写实、复杂渐变 |

## 文字问题

| 问题现象 | 根本原因 | 对策 |
|---------|---------|------|
| 模型自编台词/乱加字 | 模型倾向于补充"有用信息" | 硬约束："除指定文字外不要生成任何额外文字" |
| 中文乱码或不清楚 | 字号小或字体装饰过多 | 强调"大字号、干净字体、无花体、无小字注解" |
| 字太多读不快 | Copy Spec 阶段未压缩到位 | 回退到阶段3，每格压缩到1行短句或仅关键词 |
| 文字遮挡主体 | 未指定文字位置 | 在 prompt 中明确"文字放在对话框或画面空白区" |

## 布局问题

| 问题现象 | 根本原因 | 对策 |
|---------|---------|------|
| 格子对齐太完美 | 模型倾向于生成规整网格 | 明确写"格子位置刻意不对齐，左右上下自由错落，像手绘草稿般随意自然" |
| 格子边框歪斜倾斜 | 未约束格子边框线条的方向 | 强调"格子边框保持水平垂直"，但与"位置错位"区分开——位置可以错位，边框本身要保持笔直 |
| 格子排列太机械 | 缺乏具体的错位描述 | 给出具体错位模式："左上大格偏右，右下小格偏左"、"从左下到右上呈阶梯状"等 |
| 格子数量不对 | 模型自行决定格子数 | 在 prompt 开头明确"N格漫画"并描述具体排列方式 |
| 某个格子内容拥挤 | 该格文案或场景元素过多 | 回退到阶段2/3，拆分到更多格子或简化场景 |
| 布局过于机械死板 | 格子完全对齐网格，缺乏人性化 | 允许格子大小灵活变化、位置自然错位、间距有节奏感；关键场景可用大格子或打破边界 |
| 缺少手绘感 | 模型默认生成整齐排版 | 加入"手绘草稿感"、"像随手贴上去"、"人类创作痕迹"等描述；允许分镜线轻微波动 |

## 角色问题

| 问题现象 | 根本原因 | 对策 |
|---------|---------|------|
| 角色跨格不一致 | 每格独立生成，角色无法跨格记住 | 在每张图的 prompt 中都重复角色设定（发型/衣着/配色），并加"同一角色跨格完全一致" |
| 角色表情不变 | 未指定情绪递进 | 在每格的描述中明确角色表情（如"困惑"→"思考"→"喜悦"） |
| 多角色混乱 | 角色设定缺乏区分度 | 在设定阶段就用对比色和明显特征区分（如主角红色衬衫，配角蓝色帽子） |
