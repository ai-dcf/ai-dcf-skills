# 风格系统（阶段4必须使用）

**使用规则：** 阶段4 封装提示词时，必须同时使用两部分：
1. 本文件的 **通用约束**（所有风格共用）
2. 对应风格文件的 **风格特有要素**（质感 + 色调 + 分镜线 + 文字呈现 + 负面约束）

两部分合并后才是完整的风格定义。

## 通用约束（所有风格共用，必须写入每张图的 prompt）

### 格子布局（强调自然错位）

- **格子编排**：刻意避免规整网格对齐，格子位置自由错落，像手绘草稿般随意自然；重要格子可大幅偏离中心，小格子可偏移填补空隙；格子大小不一，主次分明，没有机械对称
- **错位布局示例**：格子可偏左偏右、上下错落；部分格子轻微旋转或倾斜；格子间距不均匀，有的地方紧凑、有的地方留出大片留白；整体看起来是"手绘感"而非"排版感"
- **人类创作痕迹**：允许格子边缘不完全平行；分镜线不必完全笔直；某些格子可有轻微重叠或边界模糊；看起来像直接在纸上随手画的草图，而非电脑生成的完美对齐
- **格子边界**：允许动态元素（动作线、效果音、角色动作、情绪线条）大幅突破格子边界，甚至在格子之间穿插，增强流动感
- **分镜节奏**：格子排布遵循视觉动线但不机械，关键场景可大幅打破布局（如跨两格的大场景、角落的小特写、倾斜的冲击格）
- **负空间**：留白区域形状不规则，有的宽有的窄，像自然呼吸的节奏
- **画风统一**：所有格子使用相同风格处理，但每个格子的边界处理和留白可以有自己的"个性"

### 角色一致性

- 同一角色跨格完全一致：发型、衣着、配色、表情特征
- 角色尺寸：同一角色在不同格子中比例相对稳定
- 视线方向：角色视线可自然引导到下一格（互动场景）

### 文字规范

- 可读性：中文必须清晰可读、无乱码；大字号
- 文字密度：每格1行短句或关键词，无密集小字/段落
- 文字位置：放在对话框或画面空白区，不遮挡主体
- 除指定文字外不要生成任何额外文字

### 视觉连贯

- 色调协调：全图主色调一致，格子间颜色和谐过渡
- 构图呼应：相邻格子间有视觉连接（同一角色延续、场景连续、动作承接）
- 焦点清晰：每格有明确主体，居中或黄金分割位
