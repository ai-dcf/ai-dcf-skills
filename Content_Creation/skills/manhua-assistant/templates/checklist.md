# 多宫格漫画出图提示词回归检查（交付前 30 秒过一遍）

## 整体结构

- [ ] 格子数量明确（2格/3格/4格/6格等），且符合"一图一场景"原则
- [ ] 布局方式已说明（手绘散落、左右错落、斜向阶梯等），格子位置**刻意不对齐**，像手绘草稿
- [ ] 格子间距与分镜线已说明（白边间隔、粗黑框、细灰线等），间距**不均匀**、可有手绘波动感
- [ ] 整体宽高比明确（如3格=3:1，4格=1:1，6格=2:3等）

## 叙事连贯

- [ ] 格子顺序与故事发展逻辑一致（起承转合）
- [ ] 相邻格子间有视觉或情节连接（同一角色延续、场景连续、动作承接）
- [ ] 每格独立讲清一个点，不堆砌内容
- [ ] 结尾格有收束感（情绪高潮/行动指引/悬念留白）

## 角色与风格统一

- [ ] 同一角色跨格保持一致（发型/衣着/配色/画风）
- [ ] 整体风格统一：奶油纸底 + 彩铅水彩 + 轻涂鸦，趣味干净
- [ ] 色调协调：全图主色调一致，避免格子间颜色冲突
- [ ] 若有多角色：主角色突出，配角不喧宾夺主

## 文字与可读性

- [ ] 每格文字控制在1行短句或关键词（无密集小字）
- [ ] 文字位置明确（对话框/画面空白区/标题区）
- [ ] 中文清晰可读、无乱码、字号够大
- [ ] 除指定文字外不加字（防止模型自编台词）

## 视觉重点

- [ ] 每格有明确焦点（人物/物体/动作）
- [ ] 主要信息居中或黄金分割位，不贴边
- [ ] 关键元素用构图强调（放大/居中/特写）
- [ ] 避免格子内元素拥挤，留白充足

## 特殊情况

- [ ] 若有情绪递进：从平静到高涨有视觉呼应
- [ ] 若有时间跨度：用天气/光线/衣着变化区分
- [ ] 若有对比冲突：使用对比色或反向构图
- [ ] 若有互动：视线/手势/物体指向下一个格子
