# 人物信息图提示词模板

## 模板结构

这是专门为**角色/人物设定信息图**设计的模板，包含头像、属性、人际关系、装备等多个信息区。

```
## 画幅 & 尺寸
Create a 16:9 landscape character infographic image

## 标题与角色名
Title (large, bold): {角色名} 角色设定卡

## 布局概览（5个主要区块）
### 中心区块：角色主画像
- Central portrait: {角色名}的[全身/半身]画像
- Setting: 穿着[服装描述]，神情[表情描述]
- Frame: 装在[相框类型]里
- Art Style: [具体艺术风格]

### 顶部区块：情绪头像展示
- Show 3-4 emotion variations: [情绪1]、[情绪2]、[情绪3]、[情绪4]
- Display style: 头像设定稿排列
- Layout: 水平排列在中心肖像上方

### 左列区块：个人档案信息
- Section title: [标题，如：个人档案 / 通缉令 / 属性面板]
- Background decoration: [背景纹理或小图案]
- Content items (3-4 entries):
  - 项目1：[具体设定信息]
  - 项目2：[具体设定信息]
  - 项目3：[具体设定信息]
- Text style: [字体风格]
- Font: Bold or decorative for important info

### 右列区块：人际关系
- Section title: [标题，如：人际关系 / 宿敌 / 羁绊]
- Show key characters: 3个关键人物
- Character display style: [展示风格，如：拍立得照片 / 悬赏令 / 宝石投影]
- Each character: [人物A（特征）]、[人物B（特征）]、[人物C（特征）]
- Layout: 垂直或网格排列

### 底部区块：装备/物品展示
- Section style: [风格，如：游戏道具图鉴 / 博物馆展品]
- Display 3 key items:
  - 物品1：[名称与特征]
  - 物品2：[名称与特征]
  - 物品3：[名称与特征]
- Arrangement: 横向并列展示
- Label: 清晰标注每个物品名称

## 整体视觉要求
- Color Palette: [颜色组合]
- Line Style: [线条风格，如：圆润 / 锋利 / 粗糙]
- Texture: [整体质感，如：复古纸张 / 高清屏幕 / 磨损海报]
- Visual Hierarchy: Clear and organized layout with balanced spacing
- Character Focus: Main character is the visual focal point

## 强制约束
Strict requirements:
- Chinese text must be clear, readable, large font
- No extra text, no watermarks, no signatures
- Sufficient whitespace, clear alignment
- No cluttered background, no random characters
- Character information must be legible at a glance
- All sections must have clear visual boundaries
- Do not add any text other than specified above
```
