# 信息图生成器

## 触发方式

当用户说类似以下内容时触发：
- "生成一张信息图"
- "给我出个信息图"
- "做一张信息图"
- "/infographic" 或 "/信息图"

---

## 流程概览

| 阶段 | 名称 | 目标 | 详细文件 |
|---|---|---|---|
| 1 | 主题输入 | 用户输入信息图的主题/主要内容 | `stages/01-topic.md` |
| 2 | 风格选择 | 智能推荐或从 16 种风格中选择一种 | `stages/02-style.md` |
| 3 | 比例选择 | 用户选择或自动匹配图片比例 | `stages/03-ratio.md` |
| 4 | 提示词生成 | 基于主题、风格和比例生成绘图提示词 | `stages/04-prompt.md` |
| 5 | 出图选择 | 用户选择：复制提示词或调用 API 出图 | `stages/05-generate.md` |

---

## 快速流程

用户只需提供一个：**信息图的主题/内容**

然后我们依次：
1. 用户输入主题 → 我复述确认
2. 我智能推荐风格或展示 16 种风格选项 → 用户选择
3. 我根据场景推荐或展示 4 种比例选项 → 用户确认或选择
4. **自动检测：** 系统自动识别是否为人物信息图，并选择合适的模板（用户无感知）
5. 我生成优化的绘图提示词
6. 用户选择：复制提示词 or 直接调用 API 生成图片

## 智能识别功能

### 人物信息图自动检测

系统会根据用户主题中的关键词自动识别是否为人物相关的信息图：

**触发关键词示例：**
- 角色类：角色、人物、人设、角色设定、角色卡、角色资料等
- 属性类：属性、性格、特征、脾气等
- 关系类：人际、关系、宿敌、伙伴、羁绊等
- 装备类：装备、物品、道具、武器等
- 表情类：表情、情绪、表情包等

**自动处理流程：**
- ✅ 检测到人物相关关键词 → 自动采用**人物信息图专用模板**
- ✅ 普通主题 → 自动采用**通用信息图基础模板**
- ✅ 用户完全无感知 → 系统自动选择，不需要用户操作

### 模板差异

| 方面 | 人物信息图 | 通用信息图 |
|-----|---------|---------|
| **布局** | 中心肖像 + 情绪 + 档案 + 关系 + 装备 | 标题 + 主体内容 + 结论 |
| **核心** | 角色全面展示 | 信息层级展示 |
| **用途** | 游戏角色卡、动画人设集、人物介绍等 | 数据图表、流程图、对比图等 |
| **结构** | 5 个信息区块 | 3-5 个内容要点 |

## 风格说明

本技能支持 **16 种专业信息图风格**，涵盖科技、商务、艺术、文化等多个领域：

### 风格系列
- **科技未来系列**（3种）：赛博霓虹、全息玻璃、工业蓝图
- **商务专业系列**（2种）：当代企业扁平、瑞士极简网格
- **高端奢华系列**（3种）：超极简暗黑、高端陶瓷浮雕、烫金浮雕纹理
- **创意艺术系列**（3种）：C4D几何晶体、抽象流体艺术、多层纸艺浮雕
- **文化风格系列**（2种）：复古钢笔手稿、孟菲斯几何波普
- **自然教育系列**（2种）：生态水彩渲染、黑板粉笔手绘
- **趣味可爱系列**（1种）：粘土拟物化Q版

### 智能推荐机制
- 如果用户明确指定风格 → 直接使用
- 如果用户未指定 → 根据主题内容智能推荐1-3个最合适的风格
- 无法智能匹配 → 展示分类风格列表供用户选择

详细风格库和推荐规则见 `references/styles.md`

## 输出规范

- 固定 1 张图、支持多种比例（由用户选择或自动适配）
  - **16:9**：PPT 展示或电脑端全屏幕展示
  - **3:4**：小红书等社交媒体发布
  - **9:16**：移动端手机全屏幕展示
  - **1:1**：通用方形格式
- 中文必须清晰可读、无乱码、大字号、少字短句
- 提示词独立代码块、便于复制
- 支持批量 API 调用（复用 image-assistant 脚本）
