# 阶段 2：风格选择（Style Selection）

**目标：** 根据用户需求智能推荐或让用户选择合适的信息图风格。

---

## 执行逻辑

### 步骤 1：检查用户是否已明确指定风格

先检查用户的输入中是否明确提到了某种风格名称，如果有，则：
- ✅ **直接采用该风格**，无需再次确认
- 输出确认语：例如"好的，将采用**赛博霓虹**风格为您生成信息图"
- **跳过后续步骤**，直接进入**阶段 3**（比例选择）

可识别的风格关键词：
- 赛博霓虹/赛博朋克/霓虹
- 全息玻璃/全息/玻璃
- 超极简暗黑/极简暗黑/暗黑模式
- C4D几何晶体/几何晶体/晶体
- 瑞士极简网格/瑞士风格/网格
- 当代企业扁平/企业扁平/扁平风格
- 高端陶瓷浮雕/陶瓷浮雕/陶瓷
- 多层纸艺浮雕/纸艺/纸艺浮雕
- 复古钢笔手稿/钢笔手稿/复古手稿
- 抽象流体艺术/流体艺术/流体
- 孟菲斯几何波普/孟菲斯/波普
- 工业蓝图设计/工业蓝图/蓝图
- 生态水彩渲染/生态水彩/水彩
- 黑板粉笔手绘/黑板/粉笔
- 粘土拟物化Q版/粘土/Q版
- 烫金浮雕纹理/烫金/浮雕

### 步骤 2：智能推荐风格

如果用户没有明确指定风格，则根据主题内容智能推荐：

1. **读取风格库**：先读取 `references/styles.md` 了解所有16种风格的详细信息
2. **分析主题内容**：识别用户主题中的关键词和场景
3. **匹配推荐规则**：根据以下规则推荐1-3个最合适的风格

#### 推荐规则速查

| 主题类型 | 关键词示例 | 推荐风格 |
|---------|----------|---------|
| 科技数据 | 科技/数据/AI/编程/黑客/未来/数字化 | 赛博霓虹 |
| 高端科技 | 界面/UI/产品/未来科技 | 全息玻璃 |
| 技术工程 | 技术/工程/架构/制造/机械 | 工业蓝图设计 |
| 商务企业 | 商务/企业/报告/PPT/专业 | 当代企业扁平 |
| 专业设计 | 系统/理性/设计规范/网格 | 瑞士极简网格 |
| 极简高端 | 极简/品牌/高端/简约 | 超极简暗黑模式 |
| 奢华精致 | 奢华/精致/优雅/品牌 | 高端陶瓷浮雕 |
| 尊贵典藏 | 尊贵/典藏/金色/奢华 | 烫金浮雕纹理 |

| 年轻活力 | 活力/年轻/时尚/潮流 | 孟菲斯几何波普 |
| 自然环保 | 自然/环保/健康/生态 | 生态水彩渲染 |

**推荐输出格式：**
```
根据您的主题「xxx」，我为您推荐以下风格：

🎯 **推荐 1：[风格名称]**（最推荐）
- 适用原因：[简要说明]

📌 **推荐 2：[风格名称]**
- 适用原因：[简要说明]

您想使用推荐1，还是推荐2，或者查看所有16种风格？
```

## 本阶段输出物

- **选定风格**：16种风格中的1种
- **确认语**：例如"好的，将采用**赛博霓虹**风格为您的「AI技术发展」信息图生成提示词"
- 进入**阶段 3**（比例选择）

---

## 注意事项

1. **优先级顺序**：明确指定 > 智能推荐 > 展示列表选择
2. **只推荐最相关的**：智能推荐时最多推荐3个风格，避免选择困难
3. **保存风格选择**：记录用户选择的风格名称，用于后续提示词生成
4. **风格详情按需加载**：只在需要生成提示词时才读取对应风格的详细描述
