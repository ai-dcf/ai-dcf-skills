# 阶段2：宫格规划（几宫格？每格讲什么？）

**目标：** 基于阶段1已确认的规格（格子数量/角色需求/用途），先"拆内容→定宫格清单→选布局"，避免一个格子塞太多导致难看难读。

## 规划原则（核心）

- **优先遵守阶段1的格子数量与角色需求**：不符合容量要求的内容，不要硬塞进一个格子。
- **一个格子=一个场景/一个核心信息**：读者扫一眼就能记住一个画面或一句话。
- **叙事连贯**：格子之间要有故事线或逻辑连接，不是孤立的图拼在一起。
- **角色一致性**：如果阶段1确认需要固定角色，同一角色在所有格子中保持一致。

## 2A：内容拆块（长文必做）

当用户给你"很长的文章/一堆段落"时，这一步就是你要做的事：

- 先按 **小标题/段落主题** 抽出模块（不要逐句改写）
- 把模块聚类成与格子数量匹配的"可画场景"（每块一句话概括，不写细节）
- 标记每块更适合的视觉形式：角色动作 / 场景变化 / 对比状态 / 情绪递进 / 结果展示

> 这一步的输出不要是提示词，而是"宫格清单草案"，让用户先选方向。

## 2B：故事线设计（常见模式）

| 模式 | 格子数 | 结构 | 适用场景 |
|------|--------|------|---------|
| 起承转合 | 4格 | 问题→过程→转折→结局 | 完整故事线 |
| 流程步骤 | 3-6格 | 步骤1→步骤2→步骤3… | 操作流程、方法说明 |
| 对比冲突 | 2-4格 | 问题→后果→解决→好处 | A/B对比、错误vs正确 |
| 情绪递进 | 3-4格 | 困惑→思考→行动→喜悦 | 体验变化、成长故事 |
| 清单呈现 | 3-6格 | 要点1→要点2→要点3… | 多个好处、多个要点 |

## 2C：画幅与布局选择

> 画幅对照表见 `references/aspect-ratios.md`（单一来源，包含所有格子数对应的比例和布局说明）

**布局原则**：强调手绘感的不规则排布，格子位置自由错位、大小不一，避免机械网格对齐。布局看起来像人类随手画的草图，而非电脑生成的完美对齐。

## 2D：角色设计（如果阶段1确认需要）

- **固定角色设定**：确定主角形象（一句话描述：年龄/发型/衣着/表情特点/配色）
- **配角设定**：如果有配角，简单定义（对比色/反差特征）
- **动作设计**：每格角色的主要动作（站立/坐着/手势/朝向/表情变化）

## 2E：风格推荐（根据内容类型自动推荐）

根据阶段1的内容类型和场景，参考 `templates/style-block.md` 中的风格对照表，推荐最合适的风格。

### 推荐逻辑

| 内容特征 | 推荐风格 | 推荐理由 |
|---------|---------|---------|
| 故事性 / 情感递进 / 角色互动 | 日式漫画风 | 动态感强，叙事张力明显 |
| 信息图 / 流程说明 / 功能对比 | 扁平插画风 | 清晰度极高，信息密度大时仍可读 |
| 文化 / 生活方式 / 情感化 | 水彩手绘风 | 有温度感，色彩柔和 |
| 技术方案 / 数据分析 / 网络安全 | 科技/暗色风 | 科技感强，信息突出 |
| 中国传统文化 / 历史 / 国学 | 水墨风格 | 意境深远，文化契合度高 |
| 怀旧 / 历史回顾 / 复古主题 | 复古印刷风 | 年代感强，故事感浓厚 |
| 教育培训 / 学术讲解 / 产品草图 | 素描手绘风 | 专业严谨，展现构思过程 |
| 游戏相关 / 复古科技 / 趣味科普 | 像素游戏风 | 复古游戏感，趣味性强 |
| 儿童内容 / 手工教程 / 节日主题 | 剪纸拼贴风 | 温暖亲切，立体手工艺感 |
| 艺术内容 / 高端品牌 / 情感渲染 | 油画质感风 | 质感厚重，视觉冲击力强 |
| 科幻内容 / 未来科技 / 潮流文化 | 赛博朋克风 | 未来感强，视觉冲击力 |
| 商务演示 / 专业报告 / 概念解释 | 极简线条风 | 极度简洁，信息传达高效 |
| 民间故事 / 社会议题 / 艺术海报 | 木刻版画风 | 力量感强，手工艺术感 |
| 通用 / 无明显特征 | 奶油纸彩铅（默认） | 暖和舒适，适合大多数场景 |

### 输出格式

```
推荐风格：[风格名称]
推荐理由：[一句话说明为什么匹配]
可选替换：[其他可选风格列表]
```

> 用户确认或切换后，风格确定。后续阶段4封装提示词时读取对应的风格文件。

## 本阶段输出物（给用户确认）

- 宫格清单（用户确认后才进入阶段3）：
  - 格子数量：N 格
  - 画幅与布局：推荐比例 + 排列方式
  - 每格：**一句话画面描述** + **文字内容**
  - 角色设定（如有）：主角形象简要描述
  - **风格选择**：确认的风格名称（阶段4 根据此项读取对应风格文件）
