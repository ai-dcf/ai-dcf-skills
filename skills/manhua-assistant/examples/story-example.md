# 示例：AI 工具选择故事（4格漫画）

这个示例展示"先规划宫格→压缩文案→输出提示词"的交付形态。

## 阶段2：宫格清单

用户确认：4格手绘散落布局，画幅 1:1，固定主角"小黑"贯穿全图

| 格子 | 一句话目的 | 画面描述 |
|------|-----------|---------|
| 格1 | 引入问题 | 小黑面对一堆工具图标，困惑表情 |
| 格2 | 尝试选择 | 小黑拿着放大镜研究，头上冒问号 |
| 格3 | 发现方法 | 小黑看到一张对比表，眼睛亮了 |
| 格4 | 获得答案 | 小黑举着正确工具，笑容灿烂 |

## 阶段3：Copy Spec

```
图编号：图1
格子数量：4格
画幅：1:1
布局：手绘散落——格子位置自由错位、大小不一，像随手画在纸上的草图
文字预算：标题1行 + 每格1行短句
角色设定：小黑，简笔画风格，黑发蓝衣

每格文案：
标题：选对AI工具
格1：工具太多，眼花缭乱
格2：一个一个试，太慢了
格3：对比表帮大忙
格4：选对工具，效率起飞
```

## 阶段4：提示词（示例）

```text
多宫格漫画，4格自然布局，画幅1:1

【风格要求】
- 奶油色纸张底（纸纹可见），彩铅线稿（笔触可见）+ 淡水彩上色（轻晕染）
- 暖色调、轻涂鸦装饰、趣味但干净
- 不要扁平矢量海报风、不要3D、不要摄影写实

【布局结构】
- 4格漫画，手绘散落布局——格子位置刻意不对齐，像随手画在纸上的草图
- 左上格偏大偏右，右下格偏小偏左，中间两格位置自由错落
- 白色间隔带，宽度不均，有的地方宽有的地方窄
- 分镜线可有轻微手绘波动感，不必完全笔直
- 格子排布遵循从左到右、从上到下的阅读动线，但不必对齐网格
- 留白区域形状不规则，有的宽有的窄，像自然呼吸的节奏

【角色一致性】
- 主角"小黑"：简笔画风格，黑发蓝衣，在所有格子中保持一致

【每格内容】
标题（居上）：选对AI工具

格1：小黑面对一堆工具图标，困惑表情，周围漂浮着AI工具图标
文字：工具太多，眼花缭乱

格2：小黑拿着放大镜研究工具，头上冒出问号
文字：一个一个试，太慢了

格3：小黑看到一张对比表，眼睛亮了，手指向表格
文字：对比表帮大忙

格4：小黑举着选中的正确工具，笑容灿烂，背景简洁明亮
文字：选对工具，效率起飞

【文字规范】
- 中文清晰可读、无乱码、大字号
- 每格1行短句，文字位置在画面空白区
- 简洁圆角矩形文字框，与画风统一
- 除指定文字外不要生成任何额外文字

【负面约束】
- 不要扁平矢量、不要3D、不要摄影写实
- 不要角色跨格不一致（发型/衣着突变）
- 不要密集小字/段落
- 不要英文与随机字符
```
